# Portfólio React

Este projeto é um portfólio pessoal desenvolvido em React utilizando Vite.

## Estrutura de Pastas

- **src/components/**: Componentes reutilizáveis do projeto.
- **src/pages/**: Páginas principais do portfólio.
- **src/assets/**: Imagens, ícones e outros arquivos estáticos.
- **src/styles/**: Arquivos de estilos globais ou temas.

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Rode o projeto:
   ```bash
   npm run dev
   ```

## Personalização

Edite os componentes, páginas e estilos conforme sua necessidade para criar seu portfólio personalizado.
